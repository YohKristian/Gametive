# API Spec

## Authentication

<!-- All API must use this authentication -->

Request :
- Header :
    - X-Api-Key : "your secret api key"

## Login

- Method : POST
- Endpoint: `/login`
- Body :

```json
{
  "username": "string, unique",
  "password": "string"
}
```

Response :

Status Code 200
```json
{
  "login": "boolean",
  "access_token": "string"
}
```

## Register

- Method : POST
- Endpoint: `/register`

## Create Product

Request :
- Method : POST
- Endpoint : `/api/products`
- Header :
    - Content-Type: application/json
    - Accept: application/json
- Body :

```json 
{
    "id" : "string, unique",
    "name" : "string",
    "price" : "long",
    "quantity" : "integer"
}
```

Response :

```json 
{
    "code" : "number",
    "status" : "string",
    "data" : {
         "id" : "string, unique",
         "name" : "string",
         "price" : "long",
         "quantity" : "integer",
         "createdAt" : "date",
         "updatedAt" : "date"
     }
}
```
